## 🌐 Nginx — הגדרה חשובה

קובץ תצורת Nginx (בשרת): `/etc/nginx/sites-available/pinkas.cloud`

חייב להכיל (לפני location כללי):
```nginx
location ^~ /ivr {
    proxy_pass http://127.0.0.1:3000;
    proxy_http_version 1.1;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
}

location ^~ /ivr-audio {
    proxy_pass http://127.0.0.1:3000;
    proxy_http_version 1.1;
    proxy_set_header Host $host;
}
```

לאחר שינוי: `sudo nginx -t && sudo systemctl reload nginx`

## 🚨 מה לעשות אם משנים שרת

1. התקן Node.js, PM2, Nginx, PostgreSQL

4. העתק `google-tts-credentials.json` לשרת

8. `pm2 start server.js --name hapinkas && pm2 save && pm2 startup`
